## 开发

```bash
# 克隆项目
git clone https://github.com/herghost000/egg-ffmpeg.git

# 安装服务端依赖
npm install

# 建议不要用 cnpm 安装 会有各种诡异的bug 可以通过如下操作解决 npm 下载速度慢的问题
npm install --registry=https://registry.npm.taobao.org

# 启动服务端
npm run dev

# 安装服务端依赖
cd client & npm install

# 启动客户端
npm run dev
```
